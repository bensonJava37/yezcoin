# yezcoin 
Year-End Token (YEZ)

Year-End Token (YEZ) is a community-driven BEP-20 token on BNB Smart Chain, designed to capture the global year-end spirit of celebration, reflection, and rewards.

Token Information
- Name: Year-End Token
- Symbol: YEZ
- Network: BNB Smart Chain (BSC)
- Total Supply: 1,000,000,000 YEZ

YEZ focuses on community participation, seasonal narratives, and open market trading.
